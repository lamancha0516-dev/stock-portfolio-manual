STEP 1

🐍 Python 환경 설치

1-1. Visual Studio Code 설치

코딩 편집기인 VS Code를 먼저 설치합니다. 코드를 보고 편집하는 작업공간입니다.

1
다운로드: https://code.visualstudio.com 접속 → "Download for Windows" 클릭
2
설치 옵션: 설치 과정에서 "PATH에 추가" 및 "탐색기 컨텍스트 메뉴에 추가" 체크박스를 반드시 체크하세요.
3
Python 확장 설치: VS Code 실행 → 왼쪽 아이콘 중 Extensions(네모 4개) 클릭 → 검색창에 "Python" 입력 → Microsoft 제공 Python 확장 설치

1-2. Python 설치

1
다운로드: https://python.org 접속 → "Downloads" → "Download Python 3.x.x" 클릭
2
⚠️ 중요: 설치 첫 화면에서 "Add Python to PATH" 체크박스를 반드시 체크한 후 Install을 클릭하세요. 이걸 빠뜨리면 나중에 명령어가 인식되지 않습니다.
3
설치 확인: 키보드 Win+R → cmd 입력 → 엔터 → 아래 명령어 입력 
python --version
Python 3.x.x가 표시되면 성공입니다.
STEP 2

📦 필수 라이브러리 설치

2-1. 라이브러리란 무엇인가?

라이브러리는 누군가 이미 만들어놓은 도구 모음입니다. 예를 들어 "엑셀 파일 읽기" 기능을 직접 수백 줄로 짜지 않아도, pandas 라이브러리를 설치하면 한 줄로 해결됩니다.

🖥️ streamlit

파이썬 코드를 웹 대시보드로 바꿔주는 도구. 이 프로젝트의 핵심 화면을 담당합니다.

📊 pandas

엑셀처럼 데이터를 처리하는 도구. 엑셀 파일 읽기/쓰기, 계산, 정렬을 담당합니다.

📈 yfinance

Yahoo Finance에서 미국 주식(QQQ, SPY 등) 실시간 가격을 가져옵니다.

📉 plotly

인터랙티브 차트(파이차트, 게이지 등)를 만드는 시각화 라이브러리입니다.

📋 openpyxl

엑셀 파일을 파이썬에서 읽고 서식(열 너비, 숫자 형식)까지 조작합니다.

📡 FinanceDataReader

한국 ETF/주식 코드를 자동으로 검색하고 가져오는 국내 전용 라이브러리입니다.

2-2. 설치 명령어

VS Code에서 터미널을 엽니다: 상단 메뉴 터미널 → 새 터미널

# 한 번에 핵심 라이브러리 모두 설치
pip install streamlit pandas openpyxl yfinance plotly

# 한국 금융 데이터 라이브러리 추가 설치
pip install FinanceDataReader

# 설치 확인 (설치된 목록 확인)
pip list
💡 팁: 설치 중 에러가 나면 pip를 먼저 업그레이드하세요: python -m pip install --upgrade pip
ℹ️ ModuleNotFoundError 발생 시: 에러 메시지에 나온 모듈명 앞에 pip install 을 붙여 설치하면 됩니다. (예: pip install requests)

📁 작업 폴더 및 엑셀 파일 준비

3-1. 작업 폴더 만들기

1
폴더 생성: 바탕화면 또는 원하는 위치에 stock 폴더를 만드세요 (경로에 한글이 없는 게 안전합니다)
예: C:\Vibe coding\stock

3-2. 나의포트폴리오.xlsx 만들기 (메인 데이터)

이 파일은 실제 보유 종목 수량을 입력하는 원본 파일입니다. 개인정보(실제 금액, 계좌번호 등)가 들어가므로 공유하지 마세요.

⚠️ 파일 이름은 반드시 나의포트폴리오.xlsx로 저장하고 stock 폴더 안에 넣어야 합니다.
ℹ️ 종목코드는 자동으로 찾아줍니다!
종목명을 입력하면 앱이 FinanceDataReader(국내 ETF), 네이버 금융 API(한국 주식), yfinance(미국 주식)를 통해 종목코드를 자동으로 검색합니다.
따라서 종목코드 열은 없어도 됩니다.
단, 신규 상장 ETF처럼 자동 검색이 안 되는 경우에만 종목코드를 직접 입력하면 됩니다.

시트 구성 예시 (계좌별로 시트를 분리)

각 계좌마다 시트를 하나씩 만듭니다. 시트 이름은 계좌 이름으로 짓습니다 (예: 국내ETF, 미국ETF, 개인연금 등)

▶ 국내ETF 시트 예시 — 최소 입력 (종목명 + 수량 + 평균매입가만 있으면 됩니다)

종목수량평균매입가
KODEX 2003035,200
TIGER 2002034,800
TIGER 미국S&P5005018,500
← 보유 종목을 계속 아래로 추가

▶ 미국ETF 시트 예시 (티커명이 곧 코드이므로 종목명에 티커를 입력)

종목수량평균매입가
QQQ5XXX.XX
SPY3XXX.XX
SCHD10XXX.XX
💡 핵심 규칙:
• 종목, 수량, 평균매입가 — 이 3개 열만 있으면 대시보드가 작동합니다
• 종목코드 열은 선택사항입니다 (자동 검색이 실패할 경우 보조용으로만 사용)
• 한국 주식/ETF는 종목 전체명 그대로 입력, 미국 주식은 영문 티커(QQQ, SPY 등)
• "My자산" 이름의 시트는 시스템이 자동으로 제외합니다

3-3. portfolio2.xlsx 만들기 (분류 정보 파일)

이 파일은 종목별로 자산구분, 테마, 한미구분을 수동으로 분류해두는 보조 파일입니다. 처음에는 빈 템플릿으로 시작해도 됩니다 — 대시보드를 처음 실행하면 나의포트폴리오.xlsx를 읽어서 자동으로 일부 채워줍니다.

ℹ️ 열 순서가 달라도 문제없습니다!
앱은 열 이름(헤더)을 기준으로 데이터를 읽기 때문에, 엑셀에서 열 순서가 본인이 사용하던 것과 달라도 됩니다.
중요한 것은 열 이름(종목, 자산구분, 테마, 한미구분, 종목코드)이 정확히 일치하는 것입니다.

▶ 직접 입력해야 하는 열 (3가지)

종목 자산구분 테마 한미구분 종목코드 (선택)
KODEX 200국내 주식국내지수KR069500
TIGER 미국S&P500미국지수추종 국내 ETF미국 시장지수US360750

SCHD미국 ETF배당성장USSCHD
← 보유 종목을 계속 아래로 추가

▶ 앱이 자동으로 채워주는 열 (대시보드 실행 시 자동 갱신)

수량평균매입가매입금액현재가평가금액손익수익률(%)시가배당률(%)연배당금
자동자동자동 자동자동자동 자동자동자동
💡 자산구분 입력 옵션: 국내 주식 / 미국 ETF / 미국 주식 / 미국지수추종 국내 ETF
💡 한미구분 입력 옵션: KR (한국 자산) / US (미국 자산)
💡 처음 시작할 때는 종목, 자산구분, 테마, 한미구분 4개 열만 만들어두면 됩니다. 나머지 열은 대시보드가 첫 실행 시 자동으로 추가합니다.

Gemini는 구글의 AI입니다. gemini.google.com에 구글 계정으로 로그인하면 무료로 사용할 수 있습니다.

4-1. 처음 시작할 때 — 기반 코드 만들기

📝 프롬프트 예시 1 — 기본 뼈대 요청
나는 Python과 Streamlit을 이용해 주식 포트폴리오 대시보드를 만들려고 해. 조건: - 파일 이름: app.py - 엑셀 파일(나의포트폴리오.xlsx)을 읽어서 종목별 수량, 평균매입가를 가져옴 - yfinance로 미국 주식 실시간 현재가 조회 - 네이버 금융 API로 한국 주식 현재가 조회 - 종목별 (현재가 × 수량 - 평균매입가 × 수량)으로 손익 계산 - 총 자산 합계, 종목별 원형 파이차트 표시 기초 코드를 만들어줘.
📝 프롬프트 예시 2 — 엑셀 시트 구조 설명 후 요청
내 엑셀 파일(나의포트폴리오.xlsx)에는 시트가 여러 개야. 각 시트 이름이 계좌 이름(예: 국내ETF, 미국ETF, 개인연금)이고, 각 시트에는 종목코드, 종목, 수량, 평균매입가 열이 있어. 이 구조를 반영해서 모든 시트의 데이터를 읽고 탭(tab)으로 계좌별로 나눠서 보여주는 Streamlit 코드를 작성해줘.
📝 프롬프트 예시 3 — 환율 연동 요청
기존 app.py에 기능을 추가해줘. 추가 기능: 1. 네이버 금융 API에서 실시간 원/달러 환율을 가져와 2. 미국 주식(QQQ, SPY, SCHD 등)의 달러 가격에 환율을 곱해서 원화로 환산 3. 사이드바에 현재 환율을 표시 4. 환율 변동 시뮬레이터: 사용자가 슬라이더로 환율 입력 → 미국 자산 평가액 변화 계산 코드 전체를 다시 완성된 형태로 써줘.

STEP 5

⚡ Antigravity로 고급 바이브코딩

5-1. Antigravity란?

Antigravity는 VS Code 안에 내장된 AI 코딩 에이전트입니다. Gemini가 대화창에서 코드를 알려주는 수준이라면, Antigravity는 실제로 파일을 직접 수정하고, 여러 파일을 동시에 분석하고, 코드를 실행까지 할 수 있는 한 단계 높은 도구입니다.

📂 파일 직접 수정

대화만 해도 app.py 파일을 직접 열고 수정합니다

🔍 코드 전체 분석

프로젝트 전체 파일을 읽고 문맥에 맞게 수정합니다

💻 명령어 실행

pip install, streamlit run 등 터미널 명령을 대신 실행합니다

🌐 브라우저 테스트

실행한 대시보드를 직접 열어서 UI를 확인합니다

5-2. 설치 방법

1
VS Code Extensions에서 설치:
VS Code 실행 → 왼쪽 Extensions 아이콘 클릭 → 검색창에 Antigravity 입력 → 설치
2
구글 계정으로 로그인:
설치 후 하단 상태바 또는 사이드바에 Antigravity 아이콘이 생깁니다. 클릭 → 구글 계정으로 로그인
3
작업 폴더 확인:
VS Code에서 stock 폴더가 열려 있는지 확인합니다. 폴더가 열려 있어야 Antigravity가 파일들을 인식합니다.

5-3. Antigravity 대화창 열기

VS Code 왼쪽 사이드바에서 Antigravity 아이콘을 클릭하면 채팅창이 열립니다. 이곳에서 한국어로 자유롭게 요청하면 됩니다.

5-4. 바이브코딩 프롬프트 예시

📝 프롬프트 예시 1 — 초기 파일 생성 요청
stock 폴더에 app.py 파일을 새로 만들어줘. 기능: - 나의포트폴리오.xlsx를 읽어서 계좌 시트별로 탭을 만들어 - 각 탭에서 종목명, 수량, 평균매입가, 현재가, 손익, 손익률(%)을 테이블로 보여줘 - 한국 주식은 네이버 금융 API, 미국 주식은 yfinance로 현재가를 가져와 - 화면 상단에는 총 매입금액, 총 평가금액, 총 손익을 큰 숫자로 보여줘 파이썬 코드를 직접 app.py에 작성해줘.
📝 프롬프트 예시 2 — 기존 코드 분석 후 버그 수정
지금 app.py를 실행하면 한국 ETF 종목의 현재가가 0으로 나와. app.py 전체를 읽고 어느 부분이 문제인지 찾아서 수정해줘. 수정 후 streamlit run app.py로 실행해서 정상 동작하는지 확인해줘.
📝 프롬프트 예시 3 — portfolio2.xlsx 자동 동기화
app.py에 기능을 추가해줘. 기능: - 대시보드가 실행될 때마다 나의포트폴리오.xlsx의 수량 변경을 감지해서 portfolio2.xlsx에 자동으로 반영해 - portfolio2.xlsx에 없는 새 종목은 맨 아래 행에 자동 추가 - 기존 종목의 행 순서는 절대 바꾸지 마 - 엑셀 서식(열 너비, 숫자 형식 #,##0)은 유지해 - openpyxl 라이브러리를 사용해서 서식을 적용해 app.py에 직접 코드를 추가해줘.
📝 프롬프트 예시 4 — 배치 파일(실행 파일) 만들기
더블클릭만 하면 대시보드가 자동 실행되는 배치 파일을 만들어줘. 파일명: 나의포트폴리오.bat 내용: - stock 폴더로 이동 - streamlit run app.py 실행 - 창이 꺼졌을 때 에러를 확인할 수 있게 pause stock 폴더에 bat 파일로 저장해줘.
📝 프롬프트 예시 5 — 성능 개선 (병렬 처리)
app.py를 보면 종목 가격을 순서대로 한 개씩 가져오고 있어. 종목이 많아지면 로딩이 너무 느려져. concurrent.futures.ThreadPoolExecutor를 사용해서 한국 주식과 미국 주식을 동시에 병렬로 가져오도록 바꿔줘. 목표: 20개 종목 기준 5초 이내에 전부 로딩되게.
📝 프롬프트 예시 6 — UI/UX 개선
app.py의 대시보드가 너무 밋밋해. 아래처럼 개선해줘. 1. 손익이 플러스면 빨간색, 마이너스면 파란색으로 테이블에 표시 (한국 증권사 스타일) 2. 총 자산 숫자 폰트를 더 크게 3. 사이드바를 두 그룹으로 나눠: 위쪽은 Market Watch(환율/VIX/공포탐욕), 아래쪽은 Tools(시뮬레이터) 4. 섹션 사이에 구분선(st.divider()) 추가 5. 페이지 탭 아이콘에 이모지 추가
📝 프롬프트 예시 7 — 특수 ETF 코드 매핑
한국 특수 ETF 중 일부는 yfinance나 일반 API로 가져올 수 없어. 예: "TIGER 코리아휴머노이드로봇산업" 같은 신규 ETF app.py에 NAME_TO_CODE 딕셔너리를 만들어서 종목명으로 직접 코드를 매핑해줘. 예시: - "TIGER 코리아휴머노이드로봇산업" → "0148J0" (네이버 금융 상품 코드) - "KODEX 미국성장커버드콜액티브" → "0144L0" 이 딕셔너리를 코드 상단에 추가하고, 현재가 조회 시 이 코드를 우선 사용하도록 해줘.
📝 프롬프트 예시 8 — 리밸런싱 도우미
사이드바에 자산 리밸런싱 도우미 기능을 추가해줘. 기능: - 슬라이더로 "목표 미국 자산 비중(%)"을 설정 (기본값 60%) - 현재 미국/한국 자산 비중을 계산해서 보여줘 - 목표 비중에 도달하려면 어떤 자산을 얼마나 더 사야 하는지 계산 - 결과를 화살표와 금액으로 명확하게 표시
💡 Antigravity 활용 팁:
• "app.py에 직접 작성해줘" — 파일을 직접 수정하게 유도
• "전체 코드를 다시 써줘"보다 "이 기능만 추가해줘"가 더 정확합니다
• 수정 후 "실행해서 확인해줘"를 붙이면 브라우저까지 열어서 검증해줍니다
• 이전 대화 내용을 기억하므로 "아까 만든 코드에 추가해줘"도 가능합니다
STEP 6

🚀 대시보드 실행 방법

방법 A — 더블클릭으로 실행 (배치 파일)

나의포트폴리오.bat 파일을 더블클릭하면 자동으로 대시보드가 실행됩니다.

💡 검은 터미널 창이 뜨면서 브라우저가 자동으로 열립니다. 터미널 창을 닫으면 대시보드도 종료됩니다.

방법 B — VS Code 터미널에서 직접 실행

# stock 폴더에서 터미널 열기 후:
streamlit run app.py

# 위 명령어가 안 되면:
python -m streamlit run app.py

방법 C — 탐색기에서 직접 실행

1
Windows 탐색기에서 stock 폴더 열기
2
상단 주소창 클릭 → cmd 입력 → 엔터
3
열린 검은 창에 streamlit run app.py 입력 → 엔터

최종 파일 구조 확인

stock/
├── app.py                  ← 메인 대시보드 코드
├── 나의포트폴리오.xlsx      ← 내 보유 종목 (개인정보 포함, 공유 금지)
├── portfolio2.xlsx         ← 종목 분류 정보 (자산구분, 테마 등)
└── 나의포트폴리오.bat       ← 더블클릭 실행 파일
